# a terminal about me
mini terminal to learn about parts of my life ☺️

### basic commands:
- `ls`
- `clear`
- `open <file>`
- `cd <folder>`

## tech stack
- react
- vite
- typescript

## run locally
`$ npm run dev`

## shoutout to the resources
- mocha color theme from [catppuccin](https://catppuccin.com)
- cat wallpaper from [wallhaven](https://wallhaven.cc/w/gpykxl)
